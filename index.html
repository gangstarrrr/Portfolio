HOLA JEJU

혼자 제주도를 가는 여행객들의 동행을 찾는 웹페이지입니다.
회원가입 / 로그인 / 게시판 / 쪽지 등의 기능이 있으며, 주로 SPRING / JSP / CSS 를 사용하였습니다.

- 회원가입 -
아이디를 입력 할 때 oninput 옵션을 사용하여 중복된 아이디가 있으면 input 박스 색을 바꾸고 submit버튼을 막습니다. 아이디와 비밀번호는 focusout 옵션을 사용해 정규표현식을 사용하여 영문과 숫자와의 조합일때만 그리고 길이는 5-12 일때만 가입이 되게 하였습니다. 프로필 사진을 선택할때는 url을 가져와서 밑에 div에 삽입될 이미지가 표현되며, upload할 폴더에 사진을 저장하여 dns값을 user table에 저장하여 수정할 때 본인의 사진이 보일 수 있게 하였습니다. 모든 값이 입력됐을때 java로 넘어가 테이블에 insert되게 하였습니다. (로그인과 회원수정도 같은 방식입니다.)

- 메세지확인 -
받은 메세지와 보낸메제지의 리스트를 확인 할 수 있습니다. 메세지가 없다면 없다는 글이 나옵니다.

- 게시판 -
테이블을 삽입하는 화면입니다. 제주도에서의 동행을 구하는 글이기에 출발 날짜와 돌아오는 날짜가 있습니다. 도착출발 날짜는 부트스트랩을 사용하여 달력에서 날짜를 선택할 수 있습니다. 입력되는 input과 보이는 input 박스가 따로 있어서 날짜형식이 아닌 다른 값이 입력되면 보내지지않습니다. 돌아오는 날짜가 출발날짜보다 빠르면 alert이 뜨며 input이 비어집니다.

- 게시판리스트 -
내림차순으로 최신 글부터 보여줍니다. 페이지는 10개씩, >>를 누르면 다음 열페이지를 보여주며 게시판 개수를 세어 거기까지의 페이지만 보여주게 만들었습니다.

- 메세지 보내기 -
게시판을 쓴 사람에게 메세지를 보낼 수 있습니다.
